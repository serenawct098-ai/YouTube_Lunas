## 六、CTA 禁用清單（防止轉化摩擦）

| ❌ 禁用行為 | ✅ 正確替代 |
|-----------|----------|
| 每條都用同一個CTA | 三條輪替一次（諮詢→礦石→IG私域） |
| Description 連結放在最底 | 兩個CTA連結必須在前3行 |
| 只講「follow我」冇具體行動 | 必須說明「DM我你出生年份」等具體指令 |
| 用「comment區」「評論區」「資訊欄」 | 用「留言區」「下面條link」「下方連結」 |
| WhatsApp連結用長網址 | 用 `wa.me/85295466508` 短連結 |
| 口播用書面語 | 旁白/TTS統一廣東話口語 |
| 字幕用廣東話口語 | 字幕統一繁體中文書面語 |
