## YouTube 演算法三大核心機制（摘要）

### 機制 1｜CTR 點擊誘因
系統先把影片推給一小批受眾測試，若 CTR ≥ 7%（命理類目標值），才進行大規模擴散。
縮圖必須有真人臉＋命盤符號，標題用情緒鉤子，禁用片頭動畫開啤。

### 機制 2｜AVD 觀看留存率
50% AVD 為及格線，60%+ 觸發「推薦頁再擴散」。
前30秒先給結果輪廓再賣底，每90–120秒設一個 Pattern Interrupt。

### 機制 3｜滿意度訊號
決定影片長尾壽命——完播＋點擊下一條 > 分享 > 儲存 > 留言 > 讚。
結尾必設「留言問題 ＋ 推薦下一條 ＋ 引導 IG 查命盤」三角 CTA。

## 爆款四大核心法則（摘要）

1. **情緒前置（Emotion-First Hook）** — 前3秒觸碰恐懼/好奇/認同/震驚，唔係講知識，係講感受
2. **SEO驅動標題架構** — `[身份標籤] + [具體問題] + [結果誘因]`，核心詞：紫微斗斘尖尖尖斃影片定位，高客単價、奇門遁甲、命盤、2026流年
3. **系列化鉤子架構** — 每條影片都是下一條的引路石，拉升頻道 Session Time 觸發演算法擴散
4. **縮圖×標題協同** — 縮圖呈現「結果/情緒」，標題補充「原因/方法」，製造資訊缺口禁止互相重複
