## 三、格式 / 風格 / 視覺規範

### 影片結構 SOP（每支標準流程）

```
0–30 秒：Hook
  → 直述問題＋核心關鍵字出現
  → 例：「化忌是什麼？為什麼你努力卻沒有回報？」
  → 禁止：Logo 動畫 / 片頭音樂 / 自我介紹

30–240 秒：結構講解
  → 命盤圖解 or 九宮格版式
  → 口播 + 畫面字幕同步核心概念

240–480 秒：案例故事
  → 感情 / 事業 / 財務 三選一場景
  → 連結觀眾痛點（第五層情緒型關鍵字）

結尾 60 秒：CTA
  → 引導到對應 Playlist（viewer journey）
  → 提示下一支影片＋IG 連結
```

### 長度標準
| 類型 | 長度 | 適用場景 |
|---|---|---|
| 主星坐命速解 | 6–8 分鐘 | 入口層，搜尋流量 |
| 四化/大限深挖 | 8–12 分鐘 | 深挖層，留存型 |
| 奇門應用案例 | 6–10 分鐘 | 應用層，決策場景 |

### 縮圖規範
- **Linara 虛擬人物半身** ＋ **命盤圖或九宮格符號**（二選一作主視覺）
- 文字：≤ 5 個字，單一痛點詞（例：精神內耗？ / 化忌踩雷？）
- 高對比色：深藍(#0D0D2B) / 深紫(#1A1A3A) × 玫瑰金色(#B4918F) / 銀白色(#E8E8F0) 文字/邊框（與 IG 美學保持一致）
- 禁止：文字堆疊、多色雜亂、無人出鏡的純文字縮圖

### 節奏規則
- 每 30–60 秒切換一次畫面視角（命盤→講解者→關鍵字特寫）
- 字幕：100% 必有，繁體中文書面語為主，英文為輔助（上中下英）
