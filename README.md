## YouTube 演算法三大核心機制（摘要）

### 機制 1｜CTR 點擊誘因

縮圖必須有真人臉＋命盤符號，標題用情緒鉤子，禁用片頭動畫開場。

### 機制 2｜AVD 觀看留存率
