## 二、口吻系統（已鎖定）

### 2.1 核心語言法則

| ❌ 禁用 | ✅ 必用 |
|---------|---------|
| 有沒有發現 | 你有冇發覺 |
| 千萬不要 | 千祈唔好 |
| 評論區 | 留言區 |
| 資訊欄 | 下面 description、下面條 link |
| 虧錢／破財 | 蝕到喊、倒錢落海、守唔到財 |
| 有客戶 | 有個客 |
| 其實 | （刪除，直接陳述事實）|
| 根本 | （刪除，改用具體描述）|
| 不是……而是…… | （禁用，改用語意自然過渡）|
| 唔係……而係…… | （禁用，同屬 AI 模板句，改用語意自然過渡）|

### 2.2 情緒助詞清單（廣東話口播 / TTS 專用）

> 適用範圍：旁白、配音、TTS 合成語音
> 每段不超過 1 個，避免堆砌

`啦` `喇` `啊` `嗯` `咦` `喎` `係呀` `囉` `咋` `哦`

> ⚠️ 注意：`喺` 係介詞（「在」之意），非情緒助詞，已從本清單移除。
> 書面語版（字幕/卡字/正文）助詞僅用：`啦` `咦` `嗯` `啊`（每段不超過1個）
> `囉` 屬廣東話口語語氣詞，書面字幕版禁用，僅限口播 / TTS。

### 2.3 反問句勾共鳴（廣東話口播專用）

- 「係咪先？」
- 「你話係唔係衰？」
- 「你有冇試過⋯⋯？」
- 「係咪好驚？」
- 「你明唔明我講咩？」

### 2.4 節奏法則

> 講完一句重點 → **停半秒** → 等觀眾個腦有畫面 → 再繼續

**節奏模板示範（廣東話口播）：**
```
「你知唔知道⋯⋯（停0.5秒）
你本命盤入面，已經寫咗⋯⋯
你今年係唔係會破財喎。（停0.5秒）
係咪覺得好驚？（停0.5秒）
唔使怕，聽我講喇。」
```

### 2.5 字幕規範（三層語言分工）

| 內容類型 | 語言 | 規範 |
|---------|------|------|
| **旁白 / 配音 / TTS** | 廣東話口語 | 情緒助詞豐富，自然口語節奏 |
| **字幕** | 繁體中文書面語 | 精準簡短，無廣東話拼音 |
| **畫面卡字 / 花字** | 繁體中文書面語 | 金色強調字 `#C9A84C`，明體 |
| **正文 / 台詞文字** | 繁體中文書面語 | 白色，宋體為主 |

- **字體**：明體／宋體為主，唔用卡通字
- **字色**：白色 ＋ 金色描邊 `#C9A84C`

## 三、視覺規範系統 v1.0

### 3.1 色彩系統

| 角色 | 色號 | 用法 |
|------|------|------|
| 主背景 | `#0D0D2B` 深海軍藍 | 永遠係底色，製造宇宙感 |
| 主字／Logo | `#B4918F` 霧玫瑰金 | 標題字、命盤符號 |

| 爆款強調 | `#C9A84C` 金色 | 重點字、邊框、星芒 |

### 3.2 縮圖構圖法則（16:9）

```
┌─────────────────────────────────────────┐
│ [左側：真人出鏡]    [右側：標題大字]  [Logo] │
│                                          │
│  人物視線→右       主標題（金色）          │
│  命盤光效          副標題（月白銀）        │
│                                          │
│ 背景：星河圖騰＋奇門九宮格淡疊加＋光霧感  │
└─────────────────────────────────────────┘
```

**構圖細則：**
- **右上角固定**：`Luna's Astral Code` Logo（透明版）
- **人物**：左側，視線望向右側文字區，製造「引導感」
- **標題**：右側，主標題金色 `#C9A84C`，副標題月白銀 `#E8E8F0`
- **背景層次**：底層星河 → 中層奇門九宮格（10-15%透明度）→ 上層光霧
- **禁用**：純白背景、純黑背景、過度濾鏡磨皮、片頭動畫 Logo 開場
