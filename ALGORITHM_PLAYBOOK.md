## 一、YouTube 三大演算法核心機制

### 1️⃣ CTR（Click-Through Rate）點擊率 —— 演算法入場券

**原理：**
YouTube 會先將影片推播給一批「種子受眾」（通常數百至數千人），根據這批人的 CTR 決定是否擴大分發。CTR 基準線：命理/靈性類 **≥5%** 為及格，**≥8%** 為優質，**≥12%** 為爆款信號。

**影響因素（權重排序）：**
| 元素 | 佔決策比重 | 說明 |
|------|-----------|------|
| 縮圖（Thumbnail） | ~50% | 最高單項影響力，3秒決定點不點 |
| 標題（Title） | ~35% | 關鍵字 + 情緒鉤子，前60字符最重要 |
| 頻道品牌辨識度 | ~15% | 粉絲認知度高→熟悉受眾CTR自然更高 |

**LINARA 適配規則：**
- 縮圖必須有真人出鏡（命理師臉孔），比純礦石/星盤圖 CTR 高約 2-3x
- 標題必含「紫微」「奇門」「2026」等高搜索意圖詞
- 禁用：純文字縮圖、過度設計導致閱讀困難的縮圖

---

### 2️⃣ AVD / Watch Time（平均觀看時長）—— 演算法留場費

**原理：**
YouTube 演算法核心指標已從「總觀看時長」進化為 **AVD（Average View Duration）** 及 **AVD%（平均觀看百分比）**。AVD% ≥ 50% = 健康，≥ 70% = 分發加速。

**關鍵機制：**
- 前 30 秒留存率決定影片是否獲擴大推播（命理內容黃金鉤：前15秒直接給「今日你的命是什麼」）
- Shorts 與長影片分開計算，互不干擾但 **Shorts 高互動可導流長片**
- 直播回放享額外 SEO 加成，同一關鍵字下優先排序

**命理頻道適配留存結構（適用15-20分鐘長片）：**
```
0:00-0:15  → 鉤子（直接給結論/衝擊數字）
0:15-2:00  → 問題放大（為何這個命局值得聽）
2:00-12:00 → 核心內容（紫微+奇門雙軌分析）
12:00-18:00 → 案例佐證（真實個案，匿名處理）
18:00-結尾  → CTA（訂閱+留言+預約link）
```

### 3️⃣ Satisfaction Signals（滿意度信號）—— 演算法續命劑

**原理：**
2025年後 YouTube 大幅升級「Post-Watch Behavior」追蹤，即看完後用戶做了什麼。這是比 Like/Comment 更深層的訊號。

**權重排序（2026年版）：**
| 信號類型 | 演算法權重 | LINARA 對應行動 |
|---------|-----------|----------------|
| 看完後繼續看頻道其他影片 | ★★★★★ | 片末設「相關播放清單」，強推系列內容 |
| 儲存影片到播放列表 | ★★★★☆ | 片中說「儲起呢條片，事後對照你嘅命盤」 |
| 分享（Share） | ★★★★☆ | 設計「你身邊一定有呢款命格」分享誘因 |
| 留言（尤其長留言） | ★★★☆☆ | 問開放式問題，如「你係邊個星曜主命？」 |
| 按讚（Like） | ★★☆☆☆ | 輔助信號，非核心 |
| 不喜歡/跳出 | 負分 | 前30秒若失去觀眾，演算法立即降權 |

**關鍵洞察：** YouTube 演算法本質係「撮合引擎」——它不推你的影片，它**推觀眾最可能看完的影片**。你的任務是讓演算法「認為」你的觀眾群是高滿意度用戶群。

## 二、爆款影片 4 大核心法則

### 法則一：「痛點前置」公式（Hook Formula）

**原理：** 命理受眾帶著明確焦慮進場（感情/事業/財運），前3秒必須直接命中這個焦慮，否則跳出。

**LINARA 黃金開場模板：**
```
版本A（衝突型）：「你2026年其實根本唔適合創業——除非你係呢款命局」
版本B（好奇型）：「我用奇門幫佢睇，發現佢今年最大貴人係佢最討厭嘅人」
版本C（數字型）：「93%紫微天機坐命嘅人都係咁死嘅——係命嗎？」
版本D（身份型）：「呢條片只係拍俾真正想改命嘅人睇」
```

**DoD（完成標準）：** 前15秒核心鉤子，無logo動畫，無片頭音樂，直入主題。

---

### 法則二：「資訊落差」結構（Curiosity Gap）

**原理：** 觀眾留下來唔係因為你說得對，係因為他們想知道「接下來是什麼」。整條影片要像一串珍珠——每個節點都留一個懸念。

**操作方法：**
- 每3-4分鐘設一個「預告鉤」：「下一part我會話你知點用奇門定位捉住你嘅貴人宮」
- 標題設「答案在片中」型：「為何天同坐命總是晚發？原因不是你想的那個」
- 章節（Chapters）功能必開，但不在前段就交代完整答案

**命理爆款選題公式（資訊落差型）：**
```
[身份標籤] + [反常識結論] + [時間urgency]
例：「天機星命格2026年大運——最多人誤解嘅一件事」
例：「你以為係桃花，其實係劫財——感情宮這個星唔係你想嘅意思」
```

---

### 法則三：「情緒-理性-行動」三段結構（ERA框架）

**原理：** 爆款影片不只給資訊，而是完成一個情緒旅程。命理內容尤其適合：觀眾帶焦慮進來，帶希望離開。

| 段落 | 功能 | 時間佔比 | LINARA應用 |
|------|------|---------|-----------|
| E（Emotion） | 共情、確認焦慮真實 | 15-20% | 「係唔係好多時感覺好努力但係搵唔到方向？」 |
| R（Reason） | 理性分析、命理解構 | 60-65% | 紫微星系分析＋奇門方位佐證 |
| A（Action） | 行動指引、轉化CTA | 15-20% | 預約全盤/下載礦石佈局指引/IG DM |

### 法則四：「SEO × 系列化」飛輪（Search + Series Flywheel）

**原理：** 單片爆款係賭博，系列爆款係系統。YouTube 搜尋佔所有觀看來源約 **40-50%**（推薦流量另計），命理類搜尋意圖強，SEO紅利大。

**LINARA 系列化架構建議：**
```
主系列A：【紫微命盤完全解讀】—— 12星曜各一集（長青搜尋流量）
主系列B：【奇門年運解析2026-2027】—— 按季度更新（時效性流量）
主系列C：【命盤×礦石佈局】—— 每個命格配對礦石（DTC轉化）
Shorts系列：每集長片剪出3條30秒Shorts（流量漏斗頂層）
```

**SEO操作標準（2026版）：**
- 標題：核心關鍵字在前40字符，例「紫微天機坐命2026大運｜...」
- 描述：前3行必含主關鍵字（YouTube搜尋索引主要讀前150字符）
- 標籤：15-20個，含「命理」「紫微斗數」「奇門遁甲」「2026運程」「香港命理」
- 章節：每個章節標題都是一個獨立關鍵字

**DoD（完成標準）：** 每條影片發布後72小時內，在目標關鍵字搜尋結果第一頁可見。
