## 一、腳本三段式結構（每條長片必用）

```
「 Hook 開場」  0–30秒
  └→ 情緒鉤子先行，再宣告影片將給出的結果/解決方案

「 主體內容」  30秒 — 至第14分鐘（或八分鐘，視影片長度）
  └→ 每 90–120 秒設一個 Pattern Interrupt（命盤截圖 / 礦石特寫 / 反常識金句）
  └→ 禁用小標題「第一點」、「第二點」，改用故事肏干擇推進

「 結尾 CTA 三角」  最後 60–90 秒
  └→ ① 留言問題（引發互動）
  └→ ② 推薦下一條影片（提升 Session Time）
  └→ ③ 引導 IG 查命盤 / 預約諦詢（DTC 轉化）
```

## 三、拍片 8 個致命地雷（絕對禁用）

| # | 地雷 | 原因 | 正確做法 |
|---|------|------|----------|
| 1 | 冗長自我介紹 | 「大家好，歡迎來到我的頻道…」 使續看率雪崩 | 直接開場白，自我介紹移至片尾 Description |
| 2 | 片頭動畫 | 3D 片頭中斷專注力，演算法降權 | 直接切入開場白，熱場 0 秒 |
| 3 | 開頭拋術語 | 「天干地支、三奇六儀」令新觀眾直接離開 | 先用白話說出結果，術語內文再解釋 |
| 4 | 標題黨狂 | 承諾奇門改運法但片中沒給出實質https | 片中必給出具體可操作干貨，再導向諮詢 |
| 5 | 語速過快像念經 | 缺乏大師氣場，降低信任感 | 語速比平時慢 1.2 倍，留空白和停頓 |
| 6 | 開頭索訂閱 | 觀眾還沒拿到價値前被索取，引發排斥 | 訂閱引導移至片尾 CTA，連同留言問題一起告知 |
| 7 | 品質粗糙網誏字幕 | 影響專業形象 | 每條影片上傳前必過字幕校對 |
| 8 | 第一條影片就把所有內容倘完 | 觀眾已經拿到，不需要訂閱了 | 每條影片給一個干貨，但留一個「更深層的鑰匙」導向下一條 |
