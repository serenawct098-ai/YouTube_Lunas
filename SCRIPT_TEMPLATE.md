## 腳本正文模板

---

### 【段落一】HOOK（0–30秒）

> **目標：** 前3秒抓住，前30秒留住。情緒/故事共鳴先行，口播自然帶出核心關鍵字。  
> **禁止：** Logo動畫、片頭音樂、自我介紹、官腔書面語開頭。

**⚑ 口吻規則：** 短句（≤20字）、口語助詞、情感問句帶入

```
【視覺提示】：____（縮圖畫面 or 開場視覺）

【口播腳本】：

（在此寫入30秒Hook腳本，參考IP_Design §5.1/5.2/5.3選擇對應主題）



【核心關鍵字首次出現時間點】：__秒
【關鍵字】：____
```

**Hook 選擇指引：**
| 主題類型 | 參考腳本 | 首句模板 |
|---|---|---|
| 化忌/能量消耗 | IP_Design §5.1 | 「在生活上有部分人，明明努力了很久，但就是感覺，有一堵牆阻礙自己一樣。」 |
| 感情/夫妻宮 | IP_Design §5.2 | 「我想問妳一個問題。妳有沒有試過——明明很喜歡一個人，但總是，差一點點。」 |
| 大限/人生轉變 | IP_Design §5.3 | 「妳有沒有感覺⋯最近身邊的能量，好像在悄悄移動。」 |

---

### 【段落二】結構講解（30–240秒，約3–4分鐘）

> **目標：** 邏輯框架骨架，拆解核心命理概念，植入行為科學比例。  
> **關鍵：** 口播＋畫面字幕同步核心概念；決策框架語言替換傳統術語。

**⚑ 口吻規則：** 每段植入1個數據（「在上百個解過的命盤當中，有X%⋯」）；禁止連續超過3句不含助詞

```
【視覺提示】：命盤圖解 / 九宮格版式 / 術語翻譯字卡

【口播腳本】：

（先解釋核心概念—用IP_Design §4.4翻譯表替換術語）



（轉折：「但妳有沒有想到——」「妳猜怎麼了⋯」）



（植入行為科學比例）



（「話說回來⋯」連結到實際應用）



【本段術語翻譯清單】：
  原術語：____  →  Linara說法：____
  原術語：____  →  Linara說法：____

【本段數據】：「在上百個解過的命盤當中，有___% 的案例⋯」
```

**結構講解段參考：**  
→ 化忌主題完整範例：IP_Design §5.4

---

### 【段落三】案例故事（240–480秒，約3–4分鐘）

> **目標：** 故事回到行為科學，讓觀眾感受並理解。三明治結構：故事→邏輯→情感共鳴。  
> **場景選擇：** 感情 / 事業 / 財務 三選一。

**⚑ 口吻規則：** 故事要有具體場景細節（不能模糊）；案例結尾必須給溫暖，不能只是診斷

```
【選用場景】：□感情  □事業  □財務

【視覺提示】：Linara出鏡說故事 / 場景示意字卡

【口播腳本】：

（引入：「我說個真實故事妳聽。」）



（故事細節：人物設定＋問題描述＋命盤發現）



（＊壓低＊：關鍵洞察點，放慢語速）



（行為科學連結：「這不是宿命，這是一個模式。」）



（數據補強：「以我諮詢過上百人的經驗來講，有X%的漏洞⋯」）



（情感收束：「這一步，就是底氣的開始。」）



【案例參考腳本】：IP_Design §5.5（化忌落財帛宮）
```

---

### 【段落四】溫暖收場＋CTA（最後60秒）

> **目標：** 先給溫暖與行動力量，再導向 CTA。禁止直接推銷跳入。  
> **CTA 順序：** 情感收束 → Playlist引導 → IG諮詢連結 → 下一支影片提示

**⚑ 口吻規則：** 收場必須有安慰句；「命盤不是用來困住妳的」類句子；CTA語氣輕柔不強迫

```
【視覺提示】：Linara溫柔表情 / 片尾卡片（End Screen）

【口播腳本】：

（情感收束，參考IP_Design §5.6 通用收場腳本）



（＊壓低，放慢＊ 最後一句力量感收尾）



（CTA 1：Playlist引導）
「如果想更深入了解___，播放清單在下方。」

（CTA 2：IG諮詢）
「想看自己的命盤，可以到 IG @lunasastralcode 找我。🌙」

（CTA 3：下一支影片提示）
「下一支，我們要聊___。」


【收場完整腳本參考】：IP_Design §5.6
```

## 上線前檢核表（腳本層）

```
□ Hook 前3秒是否用情感問句/故事開場（非自我介紹）
□ 核心關鍵字在口播前60秒是否自然出現
□ 至少1個行為科學比例數據植入（「上百個命盤中，X%⋯」）
□ 術語全部使用 IP_Design §4.4 翻譯表的決策語言
□ 是否有「＊壓低＊」標示的關鍵情緒點
□ 結尾是否有安慰句（禁止只診斷不開方）
□ CTA 是否按順序：情感收束→Playlist→IG→下一支
□ 全篇無官腔連接詞（首先/其次/總而言之）
□ 全篇無宿命語氣（「妳這輩子就是⋯」「命中注定⋯」）
□ 全篇無高飽和感嘆詞（「天啊！超準！」）
```
