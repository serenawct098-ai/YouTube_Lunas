## 一B、Shorts 三段式腳本結構（A1 類型專用，60秒以內）

> 原則：Shorts 唔係縮短版長片，係「一個觀點＋一個行動」的獨立單位。
> 語言風格：必須口語化廣東話，禁用書面語。前3秒決定完播率。

```
「衝擊鉤子」  0–3秒（硬問題 / 硬結論 / 硬指令）
  └→ 第一個字必須係「你」或直接係動詞
  └→ 禁用鋪陳、禁用問候語
  └→ 例：「今日出門前做呢一件事！」
  └→ 例：「你係唔係天機星？聽完唔好怪我。」
  └→ 例：「星期五見大客戶？等等先！」

「核心乾貨」  3–50秒（只講一個可執行指令）
  └→ 只有一個觀點 / 一個方法，唔係長片縮減版
  └→ 每10–15秒有一個視覺切換（礦石特寫 / 命盤截圖 / 字幕衝擊）
  └→ 必須有「等等，仲有更重要嘅」的懸念感推進
  └→ 例結構：「根據奇門遁甲時空盤，本週五9–11點，正西方係極強催財吉方。
             出門前配戴黃水晶，向正西方前進15分鐘，直接引爆偏財運。"

「CTA跳板」  50–60秒（唯一行動，只導一個地方）
  └→ 只導長片 或 只導IG，禁止同時叫兩個行動
  └→ 例（導長片）：「完整住宅改運佈陣，連結喺下面，長片裡教晒你。」
  └→ 例（導IG）：「想知自己財位係邊度？去IG lunasastralcode DM我。」
```

## 四、3 大高留存率錄製技巧

| # | 技巧 | 操作說明 |
|---|------|----------|
| 1 | **首句壓制語速** | 說第一句危機、懸念或指令時，語速比平常慢 1.2 倍，語調下沉，製造權威感與神秘感 |
| 2 | **眼神完全直視** | 眼神必須死死盯住鏡頭，不能有任何向左或向右看提詞機的飄移感，命理師的眼神是建立信任的第一步 |
| 3 | **節奏無縫銜接** | 講完開場白最後一句（例如：「我們直接開始 / 一定要聽完」）之後，0秒停頓，直接切入正片第一個乾貨 |

## 六、拍片 8 個致命地雷（絕對禁用）

| # | 地雷 | 原因 | 正確做法 |
|---|------|------|----------|
| 1 | 冗長自我介紹 | 「大家好，歡迎來到我的頻道…」使續看率雪崩 | 直接開場白，自我介紹移至片尾 Description |

## 九、CTA 分段植入術（高留存率版）

> 核心原則：不應等到影片最後才放 CTA。高留存影片的關鍵，是將 CTA「化整為零，分段植入」，並依 LINARA 變現飛輪，將流量精準導向「諮詢」或「實體產品（礦石）」，每次只聚焦一個核心行動。

### 📍 第一個 CTA — 影片前半段（約 5–7 分鐘處）
**目標**：引導留言互動，逼演算法推流
**適用類型**：B1 身份認同型、A2 紫微圖鑑型

```
「看到這裡，可以先暫停一下，查看自己的命盤。
如果命宮或財帛宮真的有這顆星，
在留言區留下「已報到」或者打出你的主星名字。
稍後我會在留言區抽選幾位朋友，
直接為你解答今年最需要注意的改運關鍵。」
```

**操作要點**：
- 提問要明確且易於回答（一個詞 / 一個字）
- 承諾互動但不過度（「抽選幾位」而非「每位都回」）
- 講完後0秒停頓，立刻繼續影片內容

**二選一，同一部影片只用其中一個**
