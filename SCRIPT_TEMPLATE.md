## 一、腳本三段式結構（每條長片必用）

```
「 Hook 開場」  0–30秒
  └→ 情緒鉤子先行，再宣告影片將給出的結果/解決方案

「 主體內容」  30秒 — 至第14分鐘（或八分鐘，視影片長度）
  └→ 每 90–120 秒設一個 Pattern Interrupt（命盤截圖 / 礦石特寫 / 反常識金句）

  └→ ① 留言問題（引發互動）

  └→ ③ 引導 IG 查命盤 / 預約諮詢（DTC 轉化）
```

## 四、3 大高留存率錄製技巧

| # | 技巧 | 操作說明 |
|---|------|----------|
| 1 | **首句壓制語速** | 說第一句危機、懸念或指令時，語速比平常慢 1.2 倍，語調下沉，製造權威感與神秘感 |
| 2 | **眼神完全直視** | 眼神必須死死盯住鏡頭，不能有任何向左或向右看提詞機的飄移感，命理師的眼神是建立信任的第一步 |
| 3 | **節奏無縫銜接** | 講完開場白最後一句（例如：「我們直接開始 / 一定要聽完」）之後，0 秒停頓，直接切入正片第一點乾貨 |

## 六、拍片 8 個致命地雷（絕對禁用）

| # | 地雷 | 原因 | 正確做法 |
|---|------|------|----------|
| 1 | 冗長自我介紹 | 「大家好，歡迎來到我的頻道…」使續看率雪崩 | 直接開場白，自我介紹移至片尾 Description |
| 2 | 片頭動畫 | 3D 片頭中斷專注力，演算法降權 | 直接切入開場白，熱場 0 秒 |
| 3 | 開頭拋術語 | 「天干地支、三奇六儀」令新觀眾直接離開 | 先用白話說出結果，術語內文再解釋 |
| 4 | 標題黨過激 | 承諾奇門改運法但片中沒給出實質內容 | 片中必給出具體可操作乾貨，再導向諮詢 |
| 5 | 語速過快像念經 | 缺乏大師氣場，降低信任感 | 語速比平時慢 1.2 倍，留空白和停頓 |
| 6 | 開頭索訂閱 | 觀眾還沒拿到價值前被索取，引發排斥 | 訂閱引導移至片尾 CTA，連同留言問題一起告知 |
| 7 | 品質粗糙或誤字幕 | 影響專業形象 | 每條影片上傳前必過字幕校對 |
| 8 | 第一條影片就把所有內容倒完 | 觀眾已經拿到，不需要訂閱了 | 每條影片給一個乾貨，但留一個「更深層的鑰匙」導向下一條 |
