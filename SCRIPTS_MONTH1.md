```

### 【結構講解 — 0:30–4:00】

```
好，先說基礎。

紫微斗數裡面，每個人的命盤，
有十二個宮位。
每個宮位，代表你人生的一個「部門」。

財帛宮 = 你和錢的關係。
官祿宮 = 你的事業場域。
夫妻宮 = 你的親密關係。

然後呢⋯
每一年，或每個大限，
都會有四顆星飛進你的命盤——
化祿、化權、化科、化忌。

化忌，
就是這四顆星裡面，
能量最難駕馭的那一顆。

但我要糾正一個觀念。
化忌不是「壞」。

化忌的本質，是「執著」。

化忌飛進哪個宮位，
你就會在哪個領域，
特別容易「過度用力」——
用到適得其反。

妳有沒有發現——
有些人拼命賺錢，
但賺到的錢，就是留不住。

有些人非常努力維繫感情，
但越努力，對方越想逃。

這，很可能就是化忌在運作。
```

**（配合命盤圖解畫面：顯示十二宮位 + 化忌位置標示）**

```
話說回來⋯
傳統命理師的說法是：
「化忌入財帛，錢財留不住。」

聽完，然後呢？
你能做什麼？

我的解讀方法不一樣。
我會問：

「化忌在財帛宮，

還是賺到錢就想分出去？」

這三種，解法完全不同。
但傳統說法，只給了你一句話。

這就是為什麼⋯
命盤要精確解讀，
不能只看「哪顆星在哪」。
```

### 【三層化忌分析 — 4:00–6:30】

```
好，現在進深一層。

化忌分三種：
本命化忌、大限化忌、流年化忌。

本命化忌——
是你與生俱來的「執著模式」。
基本上，是你這輩子要學習的功課。
不會消失，但可以駕馭。

大限化忌——
是你這 10 年的「主要挑戰方向」。
就像公司每十年有一個發展重心，
但這個重心帶來的是壓力點。

流年化忌——
是今年被觸發的敏感議題。
通常最明顯，也最容易被誤解成「今年運氣差」。

其實不是運氣差。
是這個議題，今年需要你去正視。

根據我解過的 200 個命盤資料——
70% 的「覺得努力沒有回報」的狀態，
背後都有一個大限化忌在官祿宮，
或者財帛宮，被觸發。

而解決方式，不是「更努力」。
是「調整方向」。
```

**（配合圖解：本命/大限/流年化忌三層對比圖）**

```

「我要理解這個系統的運作邏輯。」

優勢是——
思維快、學習快、適應快。
天機坐命的人，
幾乎可以在任何領域快速上手。

但⋯ 問題來了。

天機星的弱點，
是「完成力」。

因為你永遠覺得，
「再想想，可能有更好的方案。」

行動永遠在延遲。

天機坐命的人，

天機坐命感覺「職場有天花板」的，

三天後必須執行」的強制節點。

就是天機星最強的組合。

## 太陰坐命：你的敏感不是弱點，是被誤解的決策系統
**對應 Playlist：** P1 主星坐命全集  
**目標關鍵字：** 太陰坐命、命理心理學、精神內耗

---

### 【HOOK — 0:00–0:30】

```
太陰坐命。

你有沒有聽過別人說你：
「你太敏感了。」
「你想太多了。」
「你需要鈍感力。」

然後你開始懷疑自己——
是不是自己有問題。

其實呢⋯
問題不在你的敏感。
問題在，沒有人告訴你，
敏感是一種精密的決策系統。
```

---

### 【結構講解 — 0:30–5:00】

```
太陰星，
用現代語言說是：
「高精度情境感知能力。」

你對環境的變化，
對人際關係的微妙訊號，
對情緒的流動——
你的感知，比一般人精準很多。

這個能力，
在對的環境裡，是頂級優勢。

比如說：
談判——你能感知對方的底線。
諮詢——你能讀出客戶沒說出口的需求。
創作——你能捕捉到細節的情感層次。

但⋯ 這個能力有一個副作用。

你吸收的資訊太多。
如果沒有一個「過濾機制」，
就會處理不完，
變成精神內耗。

妳有沒有發現——
太陰坐命的人，
很容易「一個人悶著想很久」。
不是因為你不知道答案，
是因為你的資訊量太大，
需要時間整理。

這不是優柔寡斷。
是處理器太強，
需要更長的運算時間。
```

**（配合圖解：太陰星感知模型）**

```
解法呢？

太陰坐命，
最重要的事，是：
「給自己設定一個資訊邊界。」

不是所有事都需要你感知。
不是所有情緒都需要你回應。

學會說「這個不在我的感知範圍內」，
是太陰星從「精神消耗」
走向「精準決策」的關鍵。
```

---

### 【案例故事 — 5:00–6:30】

```
有個案主，太陰坐命，
在一家廣告公司做創意總監。

她說她每次開會都很累，
因為她可以感知到每個人的情緒，
但又要假裝沒感覺。

我問她：
你有沒有試過，
讓自己的感知成為一個工具，
而不是一個負擔？

她說不知道怎麼做。

我說：
下次開會，
用你的感知，
主動去讀「哪個方案讓大家真正興奮」，
而不是讀「誰不開心」。

把感知的方向，
從「防禦性感知」改成「策略性感知」。

三個月後，她說：
開會從每次都累，
變成「我現在是房間裡資訊最多的人」。
```

---

### 【CTA — 6:30–7:00】

```
總之⋯
太陰坐命，你的功課是：
「把感知從負擔，變成工具。」

你不需要變得鈍感。
你需要學會「選擇感知什麼」。

# Script 07｜7/29（週二）
## 武曲坐命：為什麼你賺得到錢，卻留不住感情？
**對應 Playlist：** P1 主星坐命全集  
**目標關鍵字：** 武曲坐命、感情走不進去、賺錢留不住

---

### 【HOOK — 0:00–0:30】

```
武曲坐命。

你的錢，
通常不差。

但你的感情，
可能是你最不知道怎麼處理的事。

為什麼？
因為武曲星，
是一顆「資源管理型」的星，
而不是「情感連結型」的星。

這不是你的問題。
是你需要了解自己的設定。
```

---

### 【結構講解 — 0:30–5:00】

```
武曲星的本質，
用現代語言說是：
「結果導向 + 資源效益型人格。」

你做事，
自然地會問：
「這樣做，效益是什麼？」
「這樣投入，回報是什麼？」

這個思維模式，
在事業和財務上，非常有效。

但⋯ 帶進感情裡，
就是一個巨大的誤解製造機。

因為感情，
本質上是不講效益的。

當你的伴侶說「我想要你陪」，
你腦子裡的第一個反應可能是：
「我明明已經賺到錢養你了，還需要什麼？」

你沒有說錯。
但感情要的，不只是資源。

妳有沒有發現——
武曲坐命的人，
在感情裡，
最難的事是：「表達情感」。

不是不愛。
是他們的表達語言，
是「行動」，是「提供」，
而不是「語言」，不是「陪伴」。

而很多人，
需要的恰恰是語言和陪伴。

這個不對頻，
就是武曲坐命在感情上最常見的卡點。
```

**（配合圖解：五種愛的語言 vs 武曲能量方向）**

```
解法呢？

武曲坐命，
不是要你改變自己的本質。
是要你：
「把「情感表達」當成一個要學習的技能，
而不是一個你不擅長的弱點。」

你學財務，你學管理，
你同樣可以學「情感表達語言」。

這是技能，不是天性。
```

---

### 【案例故事 — 5:00–6:30】

```
有個案主，武曲坐命，
做了生意很成功，
但他的另一半說要分手，
理由是「感覺不被愛」。

他說他完全不理解，
「我給她買了房，給她最好的生活，
她怎麼說不被愛？」

我說：
你的愛的語言是「提供」。
她的愛的語言是「回應」。

她需要的不是房子，
是你記得她今天說了什麼。

他說，這個他真的不擅長。

我說：那你就把它當成一個項目管理。
每天一個問題：「今天你最有趣的一件事是什麼？」
記在手機裡，下週再問一次。

他說，三個月後，他的另一半說，
「我感覺你好像換了一個人。」

他說，「我沒有換人。我換了一個溝通系統。」
```

```

你的功課是：
「學習感情的語言，就像學一門外語。」

你有能力學。
只是你之前沒人告訴你，這是可以學的。

# Script 08｜8/1（週五）
## 化忌入財帛 vs 化忌入官祿：兩種人生的卡點完全不同
**對應 Playlist：** P2 四化盲點系列  
**目標關鍵字：** 化忌落宮位、化忌入財帛、化忌入官祿

---

### 【HOOK — 0:00–0:30】

```
好，今天我們做一個比較。

同樣是化忌，
飛進財帛宮，
和飛進官祿宮——
是完全不同的卡點。

但很多人，
把這兩個搞混了。

結果用錯方法，
問題還在原地。
今天說清楚。
```

---

### 【結構講解 — 0:30–5:30】

```
先說化忌入財帛宮。

財帛宮，
代表你和「流動資金」的關係。
不是你的總資產，
是你日常和錢的互動模式。

化忌入財帛，
代表你在金錢流動上，有「執著性阻力」。

常見表現：
賺到錢，但容易在某個環節漏掉。
可能是消費模式，
可能是投資決策，
可能是定價太低。

關鍵：
化忌入財帛，
不一定是你賺不到錢——
是你在「留住錢」這件事上，
需要更有意識地管理。

然後說化忌入官祿宮。

官祿宮，
代表你的「事業場域、社會角色、工作成就」。

化忌入官祿，
代表你在職業發展上，有「方向性的阻力」。

常見表現：
努力，但升遷不順。
有能力，但被邊緣化。
或者，做了很久的工作，
忽然感覺沒有意義。

關鍵：
化忌入官祿，
很多時候是「方向問題」，不是能力問題。
你可能在做一件和你本命星不匹配的事。
```

**（配合對比圖：財帛宮 vs 官祿宮化忌行為模式表格）**

```
話說回來，
這兩種化忌，
解法是完全不同的。

化忌入財帛——
核心行動：建立財務管理系統。
不是賺更多，
是搞清楚錢去了哪裡。

化忌入官祿——
核心行動：重新定義你的「事業匹配度」。
不是更努力，
是確認方向對了再發力。

根據我解過的案例，
把這兩個搞混的人，
佔了所有「感覺努力沒有回報」的案例裡的 60%。

他們以為自己是財帛問題，
其實是官祿問題。
或者反過來。
```

化忌在官祿宮。
